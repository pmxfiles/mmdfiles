　・でっぱいモーフ　･･･　ハクの特徴（？）を再現するために導入。腕がめり込むことが多々ありますので、随時調整して下さい。
                         
  ・みっぱいモーフ　･･･　そんなものはない。

　・ボカロ目　･･･　目の色が少し薄まり、意思が薄らいだ感がでるつもり。電脳世界っぽい雰囲気を出したいときに。

　・輪郭2　…　どうしてもシャープな輪郭が好きな方へ。
