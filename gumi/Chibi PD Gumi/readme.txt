■かっつりトゥーンシェーダー
アニメみたいにバキっと塗り別けられたトゥーン表現です。
調節しやすいように色が設定できるようにしておきました。
(これはPMDEditorでトゥーンテクスチャの色を変更するのとほぼ同じです)

プリセットとして

髪.fx
顔.fx
肌.fx
服.fx

が入っています。
この中で髪と顔だけがちょっと特殊で、肌と服は設定が違うだけのものです。
髪はキューティクル用にちょっと何か入っている感じです。

これらをMMEffect→エフェクト割り当てのウインドウのmainタブから、モデルの材質に適用します。
面倒な人はモデルに服.fxでも適用すればいいと思うよ!!

※MultiEdgeについて
既に配布されているエフェクトですが、このエフェクトも同時に適用することで何かいいかんじになったりもします。
MultiEdge.pmdを読み込んだあとに、MultiEdge表情.vmdを読み込んでみてください。

☆☆☆ 注意!! ☆☆☆
何もエフェクトを弄っていない状態では、配布動画の霊夢用に調節されています。
そのまま他のモデルに適用しても「なんじゃこりゃー!!」になる可能性が高いです。
素敵にしたい場合は使いたいモデル用にfxをコピーして編集して使ってください。


◎設定項目
それぞれのfxの先頭部分に設定項目があって、色を決めます。

float ShadowStrength = 1.1;
float3 ShadowBiasColor = float3(0.3, 0.3, 0.3);
float3 ShadowBiasColorAdd = float3(-0.15, -0.15, -0.15);
float3 BaseBiasColor = float3(0.7, 0.7, 0.7);
float3 BaseBiasColorAdd = float3(0.3, 0.3, 0.3);
float SpecularStrength = 7.5;

この部分で

ShadowStrength:
　影の色の濃さを決めます。名前のわりに数字を大きくすると薄くなります。
ShadowBiasColor:
　影の色として、元の色に乗算する色を決めます。3つの数字は赤、緑、青(以下共通)の成分です。
ShadowBiasColorAdd:
　影の色を底上げする値です。プラスなら明るく、マイナスなら暗くする感じです。
BaseBiasColor:
　材質の元の色に乗算する色を決めます。
BaseBiasColorAdd:
　材質の元の色を底上げする値です。
SpecularStrength:
　髪のfxにのみ入っている、スペキュラの強度を決めるパラメーターです。

注意する点として、ShadowBiasColorAdd、BaseBiasColorAdd、SpecularStrengthは大きい値を入れすぎると白飛びする可能性が出ます。
SpecularStrength以外は基本的に0〜1の数字を入れますが、もちろん好きに調節してください。PCが壊れはしません。したらごめんなさい。

最後に、素敵なトゥーンライフが送れますように。
